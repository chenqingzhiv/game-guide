# Enshrouded 🔥

**Rise from the ashes of a fallen kingdom.** Explore a vast, open world, build your base from scratch, and defeat the monsters of the Shroud.

[![Enshrouded Banner](/assets/images/enshrouded/library_hero.jpg){: .game-hero }](/assets/images/enshrouded/library_hero.jpg)

## What is Enshrouded?

Enshrouded is a cooperative survival action RPG set in a fantasy world ravaged by a magical plague. You play as a Flameborn — the last ember of humanity, awakened to restore life to a dying realm. Mixing base building, combat, crafting, and exploration.

## Getting Started Guide

### 1. First Steps

1. Awaken at the **Flame Altar**
2. Scavenge **scrap materials** from ruined structures
3. Build a **basic shelter** and crafting stations
4. Craft **bandages** and **torches** before exploring
5. **Avoid the red Shroud** — it damages you over time

### 2. Core Progression

```
Level 1-5 (Springlands)
  → Complete the first Flame Shrine
    → Level 5-10 (Revelwood)
      → Craft Iron gear
        → Level 10-15 (Nomad Highlands)
          → Craft Steel gear
            → Level 15-25 (Kindlewastes)
              → Legendary gear & endgame
```

### 3. Essential Crafting Stations

| Station | Unlocks |
|:--------|:--------|
| Workbench | Basic tools, furniture, building blocks |
| Smelter | Ore → Ingots (copper, tin, iron, steel) |
| Charcoal Kiln | Wood → Charcoal (fuel for smelting) |
| Alchemist Station | Potions, magical upgrades |
| Loom | Plant fiber → Fabric, thread |
| Tanning Station | Animal hide → Leather |

## Combat & Skill Tips

- **Stamina management** is critical — don't spam attacks
- **Magic builds** require mana potions (stock up)
- **2-handers** deal more damage but leave you vulnerable
- **Ranged attacks** are safest — craft plenty of arrows
- **Healing potion** recipe: water + red mushroom + charcoal
- **Upgrade your Flame Altar** to unlock fast travel points

---

## Building Tips

- **Terrain can be shaped** — dig out hills, build into cliffs
- **Stone buildings** are more durable than wood
- **Place crafting stations** near each other for efficiency
- **Windows let in light** but can be broken — use shutters
- **Add a roof** to prevent rain from damaging floors
