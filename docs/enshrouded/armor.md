## Weight Classes

Light (mages/rangers): Low defense, high mobility.
Medium (hybrids): Balanced.
Heavy (tanks): High defense, low mobility.

## Best Sets

Early: Scavenger (light, +stamina) or Soldier (medium, +melee).
Mid: Nomad (ranger), Hollow (mage), Paladin (tank).
Endgame: Fell Nightmare (melee DPS), Frost Weaver (ice mage), Radiant (healer).

## Builds

Tank: Paladin set + heavy shield + mace.
Mage: Hollow set + frost wand + frost staff.
Ranger: Nomad set + longbow + daggers.
