# Enshrouded Armor Guide

Armor in Enshrouded is divided into three weight classes and multiple tiers. Choosing the right set for your build is critical for surviving Embervale's toughest encounters. This guide covers every armor set, its set bonuses, crafting requirements, and optimal builds.

---

## Weight Classes Overview

| Class | Mobility | Defense | Best For |
|-------|----------|---------|----------|
| **Light** | High (dodge cost -20%) | Low (-30%) | Mages, Rangers, Healers |
| **Medium** | Balanced | Balanced | Hybrid builds, Battlemages |
| **Heavy** | Low (dodge cost +30%) | High (+40%) | Tanks, Melee DPS |

> **Tip:** Your weight class also affects stamina regeneration speed. Light armor regenerates stamina 25% faster, heavy armor 15% slower.

## Armor Upgrades

All armor can be upgraded at the Blacksmith's workbench. Each upgrade tier requires specific materials:

| Tier | Materials Needed | Bonus |
|------|-----------------|-------|
| 1 (Common) | Resin + String | +5% Defense |
| 2 (Uncommon) | Metal Scraps + Leather | +10% Defense |
| 3 (Rare) | Metal Bars + Infused String | +15% Defense + minor stat bonus |
| 4 (Epic) | Hollow Cores + Palladium | +25% Defense + stat bonus |
| 5 (Legendary) | Dragon Scales + Shroud Crystals | +35% Defense + major stat bonus |

> **Pro Tip:** Prioritize upgrading your chest piece first — it provides the most defense per upgrade. Upgrade weapons before armor for faster clear speeds.

## Armor Tips & Tricks

1. **Mix and match for bonuses:** You need 3+ pieces of the same set to activate the set bonus. Wearing 2 pieces gives no bonus. Wearing 4+ gives the full bonus.
2. **Weight class mixing:** If you wear 2 light and 2 heavy pieces, the game averages your weight class to medium. This can be useful for tanky mages.
3. **Durability matters:** Rusty armor provides only 60% of its listed defense. Repair at the Blacksmith workbench using the same materials used to craft it.
4. **Elemental resistance:** Some endgame enemies deal elemental damage. The Frost Weaver set provides +20% ice resistance — use it against ice enemies.
5. **Visual transmog:** You can apply any armor's appearance over your current gear at the Blacksmith. Function follows the stats, looks follow the transmog.
6. **Set bonus stacking:** Set bonuses from different weight classes do stack. A Fell Nightmare chest + Eternal Wanderer legs + Fell Nightmare helm + Fell Nightmare gloves still gives the full Fell Nightmare set bonus (4 pieces).
7. **Upgrade before replacing:** A Tier-4 Scavenger set (level 1-10) can match Tier-1 Nomad (level 11-20) in defense. Don't replace upgraded gear immediately.
