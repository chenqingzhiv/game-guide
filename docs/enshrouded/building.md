# 🏗️ Enshrouded — Building & Base Construction Guide

*Last updated: July 2026 | Game version: 1.0+*

---

Enshrouded's building system is one of its biggest selling points — **full voxel-based construction** with terrain shaping, structural integrity, and a deep material system. Unlike games where you snap pre-made structures together, Enshrouded lets you carve into cliffs, raise mountains, and build sprawling castles block by block. This guide covers everything from your first wooden shack to an impenetrable fortress.

---

## Building System Overview

| Feature | Description |
|:--------|:------------|
| **Voxel Construction** | Every block is individually placeable — full creative freedom |
| **Terrain Shaping** | Shovel, pickaxe, and terrain tool let you dig, raise, and flatten |
| **Structural Integrity** | Blue (stable) → Green (okay) → Red (unsafe) color coding |
| **Flame Altar** | Defines your building territory — upgrade to expand |
| **Comfort System** | Furniture and decorations give resting buffs |
| **Material Tiers** | Wood → Stone → Glass → Luminescent blocks |

## Flame Altar & Building Territory

The **Flame Altar** is the heart of every base. You must place one before you can start building. It defines the buildable area and serves as your respawn point.

### Altar Levels

| Level | Radius | Unlock Cost | Requirements |
|:------|:-------|:------------|:-------------|
| 1 | 30m | None | 5 Stone + 3 Wood |
| 2 | 40m | 10 Shroud Essence | Crafting Station nearby |
| 3 | 55m | 25 Shroud Essence + 5 Ancient Cores | Alchemist Station |
| 4 | 70m | 50 Shroud Essence + 10 Ancient Cores | 3 NPCs rescued |

> 💡 **Tip:** Place your first altar in a central, flat area near copper and stone deposits. You can have **multiple altars** per world — use secondary altars for outpost bases near boss arenas or resource nodes.

### Multiple Altars

| Number of Altars | Total Cost (cumulative) |
|:-----------------|:------------------------|
| 2 altars | +50% Shroud Essence cost |
| 3 altars | +100% Shroud Essence cost |
| 4 altars | +150% Shroud Essence cost |

Each additional altar costs more Shroud Essence. For most players, **2 altars** is the sweet spot — one main base and one forward outpost.

## 🏔️ Terrain Shaping (Voxel System)

Enshrouded's voxel terrain system is unique among survival games. **You can modify the terrain itself**, not just place structures on top of it.

### Tools

| Tool | Unlock | Function | Best Use |
|:-----|:-------|:---------|:----------|
| Shovel | Crafting Station | Dig soil, sand, and loose earth | Foundation prep, moats |
| Pickaxe | Starting tool | Mine stone, ore, and terrain blocks | Tunnels, underground rooms |
| Terrain Shaper | Carpenter Table | Raise, lower, and flatten terrain | Landscaping, platforms |
| Build Hammer | Starting item | Place/remove building blocks | All construction |
| Axe | Starting tool | Chop trees (no terrain effect) | Clearing building sites |

### Terrain Shaping Techniques

#### Digging & Excavation

- **Dig down** to create basements, underground rooms, and hidden passages
- **Dig into cliffs** — build cliffside homes with natural stone walls
- **Dig a moat** around your base — enemies can't path over deep holes
- **Maximum dig depth:** ~15 blocks below the original terrain surface

#### Raising Terrain

- **Raise platforms** for elevated building sites (great for castles on flat ground)
- **Create earth ramps** to access upper levels without stairs
- **Fill in holes** to create flat building surfaces
- **Build artificial hills** for scenic landscaping

#### Flattening

- **Flatten tool** creates perfectly level surfaces — essential for foundation stability
- **Auto-snap to existing structures** — flatten connects to your building's floor level
- **Best practice:** Flatten a 5×5 area before placing your Flame Altar for a level start

### Underground Building Examples

| Build Type | Depth | Techniques |
|:-----------|:------|:-----------|
| Basement | 2–3 blocks down | Dig out, line with stone walls, add stairs |
| Underground bunker | 5–8 blocks down | Reinforce ceiling with stone pillars |
| Cliffside home | Surface level | Horizontal dig into cliff face, glass front wall |
| Cave base | Natural cave | Use existing cave, add floors and walls |
| Subterranean hall | 10+ blocks down | Large pillars every 8 blocks to prevent collapse |

> ⚠️ **Structural Collapse:** Deep underground rooms need support pillars every 8 blocks or the ceiling will show **red** structural integrity (collapsing). Always test by removing your support scaffold — if it turns red, add more pillars.

## 🔵 Structural Integrity System

Enshrouded uses a **color-coded structural integrity** system to show how stable your building is.

### Color Codes

| Color | Meaning | Risk |
|:------|:--------|:-----|
| 🔵 **Blue** | Perfectly stable | None — fully supported |
| 🟢 **Green** | Stable but loaded | Moderate — can add more weight |
| 🟡 **Yellow** | Near limit | High — one more block may collapse |
| 🔴 **Red** | Unstable | **Will collapse** — remove weight or add support |
| ❌ **Falling** | Collapsing | Blocks break on impact |

### Support Mechanics

- **Every block must connect to the ground** (directly or through other blocks)
- **Walls transfer load upward** — you can stack many blocks on a wall
- **Floors have a max span** of 8 blocks without support below
- **Pillars reset load** — place a pillar every 6–8 blocks for long spans
- **Overhangs** (blocks without support below) have strict limits:
  - Wood: max 2 blocks overhang
  - Stone: max 3 blocks overhang
  - Stone Brick: max 4 blocks overhang

### Support Tips

```
Good:   ██ pillar every 6 blocks under floor spans
Bad:    ██................██ (long unsupported span = red)
Better: ██...██...██...██ (pillars every 6 blocks = blue)
```

1. **Build from ground up** — each floor should rest on walls below
2. **Use stone pillars** for multi-story buildings (wood pillars are weaker)
3. **Archways distribute weight** — placing arches over doorways strengthens the wall above
4. **Don't remove support scaffolding** until the replacement structure is complete
5. **Reinforced corners** — corners with two intersecting walls are 2× stronger

## Roof & Stair Construction

### Roof Types

| Roof Type | Material | Slope | Weight | Cost |
|:----------|:---------|:------|:-------|:-----|
| Wood Roof | Wood | 45° | Light | 6 Wood Logs per section |
| Stone Roof | Rough Stone | 30° | Medium | 8 Stone per section |
| Clay Roof | Limestone | 30° | Medium | 6 Limestone + 3 Clay |
| Slate Roof | Stone Brick | 20° | Heavy | 10 Stone Blocks per section |
| Glass Roof | Glass | Any | Light | 6 Glass + 2 Iron Frames |

### Roof Building Tips

- **Build scaffolding first** — place temporary pillars to support roof blocks during construction
- **Sloped roofs look best** with a 1-block overhang at the edges
- **Glass roofs** let in natural light but don't block rain — use them over enclosed rooms only
- **Roofs count toward structural load** — heavy stone roofs need extra support pillars on lower floors
- **Dormer windows** can be added by leaving gaps in the roof and framing with glass blocks

### Stair Types

| Stair | Material | Width | Rotation |
|:------|:---------|:------|:---------|
| Wood Stairs | Wood | 1 block | 90° turn possible |
| Stone Steps | Rough Stone | 2 blocks | Straight only |
| Spiral Stairs | Wood + Stone blocks | 1 block | Full 360° |
| Grand Staircase | Stone Brick + Decorative | 3+ blocks | Custom |
| Ladder | Wood | 1 block | Vertical only |

### Spiral Staircase Construction

To build a spiral staircase:

1. Place a **central pillar** (stone, 1×1)
2. Place stairs around it in a **clockwise pattern**, 1 block higher each step
3. Remove the central pillar after all stairs are placed (optional)
4. Add railings on the outer edge for aesthetics

### Optimizing Comfort

**Maximum comfort per zone:**

- **Combined great hall:** All of the above in one room = 39+ comfort

> 💡 **Tip:** You only need one room with high comfort — the Rest Bonus applies globally to your character. Build a single great hall with all comfort items rather than spreading them across rooms.

## 🏰 Advanced Castle Construction

### Multi-Story Foundations

```
Floor 3: Roof + lookouts
Floor 2: Bedrooms + crafting
Floor 1: Great hall + storage + kitchen
Ground: Entrance + defense + animal pens
```

1. **Base foundation:** 3-block-thick stone floor (prevents enemies digging in)
2. **Walls:** 2-block-thick stone walls (with internal hollow core for wiring/storage)
3. **Roof:** Sloped stone with glass skylights
4. **Defense:** Arrow slits on each floor, elevated battlements

### Castle Layout Template

### Tower Construction

- **Corner towers:** 5×5 base, narrow to 3×3 at top for battlements
- **Gate towers:** Flank your main entrance with two 3×3 towers
- **Watchtowers:** 3×3 base, tall (8+ blocks), with roof access
- **Separation:** Towers should be structurally separate from walls (they're stronger as independent units)

### Underground Complex

Underground bases are viable but require planning:

## 💡 Building Tips & Tricks

### Lighting Strategies

| Light Source | Radius | Fuel | Best Place |
|:-------------|:-------|:-----|:-----------|
| Torch | 4 blocks | Wood | Walls, corridors |
| Wall Sconce | 5 blocks | Oil | Interior walls |
| Standing Lantern | 6 blocks | Oil | Floors, gardens |
| Chandelier | 8 blocks | Oil + Wick | Great halls |
| Luminescent Block | 6 blocks | None (permanent) | Subterranean, decoration |
| Fireplace | 10 blocks | Wood | Living areas |
| Glowing Shroud Bloom | 3 blocks | None | Decorative, natural look |

### Space-Saving Tips

- **Vertical storage:** Stack chests on different floors, not all on ground level
- **Wall-mounted stations:** Some crafting stations fit on 1-block-wide walls
- **Half-block tricks:** Use half-height blocks for counters, window sills, and railings
- **Integrated storage:** Build storage into your walls (hollow core) to save floor space
- **Multi-purpose rooms:** Kitchen + dining in one room, crafting + storage in another

### Aesthetic Tips

1. **Color variation:** Alternate between 2–3 material types for visual interest
2. **Window placement:** Evenly spaced windows (every 3–4 blocks) look most natural
3. **Roof overhangs:** 1-block overhang on all roof edges creates depth
4. **Landscaping:** Shape terrain around your base — stairs up the hill, planted trees
5. **Lighting layering:** Combine ceiling lights + wall sconces + floor lamps for warm light
6. **Interior archways:** Frame doorways with arch blocks for a polished look
7. **Symmetry is optional** — asymmetrical builds can look more natural

### Performance Optimization

| Issue | Cause | Fix |
|:------|:------|:-----|
| Low FPS near base | Too many light sources | Replace torches with luminescent blocks |
| Lag when placing blocks | Too many physics checks | Use smaller rooms (20×20 max) |
| Slow load time | Massive underground complex | Surface buildings load faster |
| Stuttering | Too many NPCs/items | Keep decorative items under 200 per room |
| Freeze on terrain edit | Large terrain modification | Modify terrain in 10×10 sections |

## Common Building Mistakes

| Mistake | Result | Fix |
|:--------|:-------|:-----|
| Building too large too fast | Structural collapses, wasted materials | Start small (5×5), expand gradually |
| Ignoring structural integrity | Roof collapses on you | Check colors — add pillars if yellow |
| Placing altar on uneven ground | Crooked floors and walls | Flatten the area first |
| No roof on storage | Rain damages items | Add roof before storing valuables |
| Single-thick exterior walls | Enemies break in easily | Double-layer your outer walls |
| Basement without pillars | Cave-in during combat | Add pillars as you dig |
| Too many torches | Performance lag | Switch to luminescent blocks |
