# Subnautica Materials & Resources Database

**Everything you can mine, collect, and craft on Planet 4546B.** From common Titanium to rare Kyanite — find out where to get what, what tool you need, and what it's used for.

> 💡 **Quick Tip:** Limestone Outcrops are your best early source of Titanium and Copper. Smash every one you see — Titanium is the backbone of nearly every build in the game.

## 🪨 Outcrop Source Table

Resources are collected from breakable outcrops. Different outcrop types yield different materials:

| Outcrop Type | Common Biomes | Drops |
|:-------------|:--------------|:------|
| **Limestone Outcrop** | Every biome | Titanium (50%), Copper Ore (50%) |
| **Sandstone Outcrop** | Grassy Plateaus, Kelp Forest, Safe Shallows | Silver Ore (45%), Gold (35%), Lead (20%) |
| **Shale Outcrop** | Mountains, Lost River, Jellyshroom Caves, Inactive Lava Zone | Diamond (30%), Gold (30%), Lithium (25%), Lead (15%) |
| **Basalt Outcrop** | Inactive Lava Zone, Lava Lakes | Kyanite (40%), Ruby (30%), Diamond (30%) |

## 📊 Tool Requirements for Mining

| Tool | What It Can Harvest | Crafting Recipe |
|:-----|:--------------------|:----------------|
| **Hand (no tool)** | Limestone/Sandstone outrops, Quartz, Salt, flora | None |
| **Survival Knife** | Flora, Creepvine, harvest small resources | 1x Titanium + 1x Silicone Rubber |
| **Laser Cutter** | Cut through sealed doors (wrecks) | 1x Battery + 1x Diamond + 1x Titanium |
| **Prawn Drill Arm** | Large resource deposits, Kyanite, Nickel | 2x Diamond + 2x Titanium |
| **Propulsion Cannon** | Move/loot objects, collect loose items | 1x Plasteel Ingot + 1x Wiring Kit + 1x Magnetite |

## 📈 Resource Progression Path

| Game Phase | Objective | Key Resources Needed | Recommended Vehicle |
|:-----------|:----------|:---------------------|:-------------------|
| **Early Game** | Build habitat & survive first days | Titanium, Quartz, Copper, Acid Mushroom | None |
| **Early-Mid** | Craft Seamoth, explore to 200m | Lead, Lubricant, Silver, Gold | Seamoth |
| **Mid Game** | Upgrade Seamoth MK1-MK2, reach 500m | Magnetite, Ruby, Plasteel Ingot | Seamoth MK2 |
| **Late Game** | Build Prawn Suit, explore Lost River | Aerogel, Diamond, Nickel, Kyanite | Prawn Suit + Cyclops |
| **End Game** | Build Neptune Rocket, finish story | Kyanite, Ion Cube, Plasteel, Nickel | Prawn Suit MK1 |
| **Post-Game** | Base expansion & decoration | All resources — unlimited building | Cyclops MK2+ |
