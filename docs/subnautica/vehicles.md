# Subnautica Vehicles Guide

**Every vehicle you can build on Planet 4546B — from the nimble Seamoth to the rocket that takes you home.** This guide covers vehicle stats, depth module upgrades, crafting recipes, and upgrade modules.

> 💡 **Quick Tip:** Always craft the next depth module before venturing into deeper biomes. Getting crushed at 900m in a Seamoth that's only rated for 500m is a save-ending mistake.

---

## 🚗 Seamoth

The Seamoth is your first and most agile vehicle — a fast, single-passenger submersible perfect for exploration.

| Stat | Value |
|:----|:------|
| **Crafting** | 1x Mobile Vehicle Bay + 1x Power Cell + 1x Titanium Ingot + 1x Lubricant + 1x Lead |
| **Health** | 200 HP |
| **Max Depth (Base)** | 200m |
| **Speed** | 11.0 m/s |
| **Storage** | 48 slots (4 built-in + upgrade module) |
| **Power** | 1x Power Cell (200 energy) |

### Depth Module Upgrades

| Module | Max Depth | Materials | Fragment Location |
|:-------|:----------|:----------|:------------------|
| **MK1** | 300m | 1x Plasteel Ingot + 1x Enameled Glass + 1x Wiring Kit | Kelp Forest, Grassy Plateaus |
| **MK2** | 500m | 1x Plasteel Ingot + 1x Enameled Glass + 1x Ruby + 1x Magnetite | Jellyshroom Caves, Underwater Islands |
| **MK3** | 900m | 1x Plasteel Ingot + 1x Enameled Glass + 2x Nickel Ore + 1x Kyanite | Lost River |

### Seamoth Upgrade Modules

| Module | Effect | Materials |
|:-------|:-------|:----------|
| **Storage Module** | +48 storage slots | 2x Titanium + 1x Silver Ore |
| **Perimeter Defense** | Electric shock to repel creatures | 1x Wiring Kit + 1x Battery + 1x Copper Wire |
| **Sonar** | Ping to reveal terrain (uses power) | 1x Wiring Kit + 1x Magnetite + 1x Advanced Wiring Kit |
| **Torpedo System** | Launch gas/vortex torpedoes | 1x Wiring Kit + 2x Titanium |
| **Hull Reinforcement** | +25% damage reduction | 1x Plasteel Ingot + 2x Lithium |

| Stat | Value |

## 🔧 Vehicle Bay Crafting

All vehicles require a **Mobile Vehicle Bay** to build:

```text
Mobile Vehicle Bay → 1x Titanium Ingot + 1x Lubricant + 1x Computer Chip + 1x Power Cell
```

Deploy the bay on open water (not inside a base or cave), then interact with the hologram console to build your vehicle.
