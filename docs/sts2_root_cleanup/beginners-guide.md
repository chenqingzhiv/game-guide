## 核心技巧总结

### 1. 先防守，再进攻
不要每回合把所有攻击牌都打出去。**Block 比伤害更重要**——活着才有输出。

### 2. 跳过卡牌奖励
新手最容易犯的错：每次打赢都选一张卡。**20-30 张的精简卡组 > 40 张的臃肿卡组**。如果三张都不适合你的 Build，果断 Skip。

### 3. 商店优先删牌
Strike 和 Defend 是卡组里最弱的牌。75 金币删一张 = 整个 Run 最好的投资之一。

### 4. 血量健康就打 Elite
Elite 给遗物——遗物是改变 Run 走向的关键。血量够就主动找 Elite 打，别绕路。

### 5. 优先升级 > 休息
火堆前：升级一张核心卡 > 回血。血量是资源，升级是永久提升。

### 6. 规划路线
开始前扫一眼地图：火堆位置、Elite 分布、商店顺序。别盲目选路。

### 7. 阅读敌人意图
每回合敌人头上的图标告诉你下回合做什么：🗡️攻击 / 🛡️防御 / 🌪️ Buff。养成每回合看意图的习惯。

### 8. 药水不用省
药水不是在 Boss 战才用的。精英战、甚至普通战血量危险时就喝——省下的 HP 可能在下一场救你。

## Ironclad 新手 Build 推荐

视频演示了两套适合新手的 Ironclad 构筑：

### 🛡️ Body Slam 防御流
- **核心**: 堆 Block → Body Slam 转化为伤害
- **关键卡**: Impervious, Flame Barrier, Shrug It Off, Barricade
- **遗物**: Bronze Scales, Oddly Smooth Stone

### 💪 Strength 流
- **核心**: 叠 Strength → 多段攻击
- **关键卡**: Ashen Strike, Brand, Burning Pact, Fiend Fire
- **遗物**: Charon's Ashes, Toasty Mittens
