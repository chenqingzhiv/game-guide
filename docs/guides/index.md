# 攻略指南

种田游戏通用技巧汇总。

---

## 🔰 新手入门

1. **先看攻略再动手** — 了解作物收益再种地
2. **攒钱优先于升级** — 第一年先赚够本金
3. **钓鱼是前期最佳收入** — 大部分农场游戏前期钓鱼最赚钱
4. **不要扩建太快** — 工具升级比买新地划算

---

## 💰 通用赚钱公式

```
日收益 = （售价 - 种子成本）÷ 生长天数
```

## 📚 常见问题

**Q: 第一年冬天干什么？**
A: 下矿、钓鱼、整理农场、升级工具、刷好感度。

**Q: 什么时候该升级工具？**
A: 当浇水/砍树占用超过半天时间时。

**Q: 动物值得养吗？**
A: 前期不推荐（投入大回本慢），中期稳定收入后可以考虑。
