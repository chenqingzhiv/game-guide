## What is Rust?

Rust is a multiplayer survival game where you start with nothing but a rock and a torch. Gather resources, build bases, raid enemies, and survive against 100+ other players on each server. **Monthly wipes** keep the game fresh and competitive.

### Key Game Systems

| System | Importance | Details |
|:-------|:-----------|:--------|
| **Tool** | Essential | Craft tools, weapons, armor — progress from stone to metal to HQM |
| **Building** | Core | Walls, doors, honeycombing, TC — your base is everything |
| **Farming** | Mid-game | Grow food, manage genetics, create teas |
| **Monuments** | Loot | Military Tunnels, Launch Site, Cargo Ship — best loot spawns |
| **Raid** | End-game | Explosives, rockets, raid towers — loot other players |
| **PvP** | Constant | Gunplay, positioning, team coordination |
