## What is Rust?

Rust is a multiplayer survival game. Start with nothing, gather resources, build bases, survive.

## Getting Started

1. Pick a 2x or 3x modded server
2. Spawn away from monuments
3. Stone tools, 2x1 base, sleeping bag
4. Day 1 goal: Secure base with door and lock
