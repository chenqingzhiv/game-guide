# 🏛️ Rust Monuments Guide

> Data version: Rust 2026 | Last updated: 2026-07

Monuments are the heart of Rust. They contain the best loot, the toughest NPC scientists, and the most intense PvP combat. This guide covers every monument, its loot tables, difficulty, and optimal farming strategies.

---

## 🗺️ Monument Tiers

### Tier Breakdown

| Tier | Loot Quality | Scientist Difficulty | PvP Activity | Examples |
|:-----|:-------------|:--------------------|:-------------|:---------|
| **Tier 1** | Low/medium | None or easy | Low | Lighthouse, Supermarket, Gas Station |
| **Tier 2** | Medium/high | Medium | Medium | Airfield, Trainyard, Water Treatment |
| **Tier 3** | High/very high | Hard + radiation | High | Launch Site, Military Tunnels, Arctic Research |
| **Tier 4** | Max tier loot | Very hard (heavies) | Extreme | Cargo Ship, Oil Rig (Large), Underwater Labs |

## 🏪 Tier 1 Monuments

### Gas Station ⛽

| Stat | Value |
|:-----|:-------|
| Location | Roads, near towns |
| Radiation | None |
| Scientists | 0 |
| Best loot | Food boxes, low-tier crates |
| PvP risk | Low (usually contested by nakeds only) |
| Loot run time | 2 minutes |

**Strategy:** Quick hit for food and basic components. Good for fresh spawns.

### Supermarket 🛒

| Stat | Value |
|:-----|:-------|
| Location | Roads |
| Radiation | None |
| Scientists | 0 |
| Best loot | Food, barrels, tool boxes |
| PvP risk | Low |
| Loot run time | 3 minutes |

**Strategy:** Check the back room for crates. Use the roof for elevation.

### Lighthouse 🔦

| Stat | Value |
|:-----|:-------|
| Location | Coastline |
| Radiation | None |
| Scientists | 2 (rooftop) |
| Best loot | Military crate (top floor), tool boxes |
| PvP risk | Low–medium |
| Loot run time | 5 minutes |

**Strategy:** Kill the 2 rooftop scientists (they're weak), loot the military crate, check the basement for barrels.

## 🏭 Tier 2 Monuments

### Airfield 🛩️

| Stat | Value |
|:-----|:-------|
| Location | Inland (medium map) |
| Radiation | Low (need hazmat or rad suit for hangars) |
| Scientists | ~8 |
| Best loot | 2 military crates, elite crate, 6+ green crates |
| PvP risk | Medium |
| Loot run time | 10 minutes |

**Loot Run Route:**
1. Enter from the gate → control tower (2 green crates + military crate on roof)
2. Drop to hangar 1 (3 green crates, 1 military crate)
3. Hangar 2 (2 green crates)
4. Back rooms (elite crate in locked room — need fuse)
5. Exit

### Trainyard 🚂

| Stat | Value |
|:-----|:-------|
| Location | Inland |
| Radiation | Low |
| Scientists | ~10 (some heavies) |
| Best loot | 1–2 military crates, 6+ green crates |
| PvP risk | Medium–high (open sightlines) |
| Loot run time | 12 minutes |

**Strategy:** Use train cars for cover. Watch the rooftops — campers love them. Check the yellow crane for a hidden military crate.

### Water Treatment Plant 💧

| Stat | Value |
|:-----|:-------|
| Location | Inland |
| Radiation | Medium (hazmat needed in center) |
| Scientists | ~12 |
| Best loot | 2 military crates, elite crate, 8+ green crates |
| PvP risk | High (complex layout = ambushes) |
| Loot run time | 15 minutes |

**Strategy:** Enter from the highest point (water tower) to spot enemies. The underground tunnels have great loot but are deathtraps.

## 🚢 Tier 4 Monuments

### Cargo Ship 🚢

| Stat | Value |
|:-----|:-------|
| Location | Ocean (spawns randomly, rotates map) |
| Radiation | High |
| Scientists | ~30 (heavies + lots of scientists) |
| Best loot | 2 elite crates, 3+ military crates |
| PvP risk | Extreme |
| Loot run time | 15 minutes (before it despawns) |

**Strategy:**
1. Build a boat or mini-copter to reach it
2. Climb ladder at front of ship
3. Clear deck → work down through rooms
4. Watch the bridge (sniper-heavy area)
5. Loot elite crates in the captain's quarters
6. **Escape** — use boat/copter to get out fast (others will see the ship docked)

### Oil Rig (Large) 🛢️

| Stat | Value |
|:-----|:-------|
| Location | Ocean (fixed positions) |
| Radiation | High |
| Scientists | ~15 + 2 heavies + timed crate |
| Best loot | 2 elite crates, 3+ military crates, timed crate |
| PvP risk | Extreme |
| Loot run time | 20 minutes (timed crate + 30 min) |

**Strategy:**
1. Boat to rig, tower snipe scientists before climbing
2. Bring **red keycard** for locked room
3. Clear bottom → middle → top crane
4. Activate timed crate (30 min) — defend it or leave and come back
5. Boat escape is vulnerable — counter-raiders will try to take your boat

### Small Oil Rig 🛢️

| Stat | Value |
|:-----|:-------|
| Location | Ocean |
| Radiation | High |
| Scientists | ~8 + 1 heavy |
| Best loot | 1 elite crate, 2 military crates |
| PvP risk | High |
| Loot run time | 10 minutes |

## ⚠️ General Monument Tips

| Tip | Why |
|:----|:-----|
| **Visit at low-pop times** (6–9 AM server time) | Less contest, safer farming |
| **Bring a jackhammer** | Break into locked rooms without keys |
| **Use drop boxes** for quick extraction | Call a teammate with mini to airlift loot |
| **Don't go deep** into a monument without scouting | You'll get trapped |
| **Wear a hazmat suit** — not just roadsign | Radiation kills quickly in T3+ |
| **Bring med syringes** (4–6 per run) | Scientists hit hard |
| **Avoid the center of T3 monuments** during peak hours | Campers will kill you |
