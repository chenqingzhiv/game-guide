### Recoil Control Basics

| Weapon | Pattern | Vertical | Horizontal | Difficulty |
|:-------|:--------|:---------|:-----------|:-----------|
| AK-47 | Moderate | Pull down consistently | Slight left-right drift | ★★★★★ |
| LR-300 | Easy | Minimal pull | Almost none | ★★☆☆☆ |
| Thompson | Easy | Slight pull | Random spread | ★★★☆☆ |
| Custom SMG | Moderate | Up-left drift | Wobble | ★★★☆☆ |
| MP5 | Very easy | Almost none | Minimal | ★☆☆☆☆ |

### AK-47 Spray Training

```
Bullet 1–5:  Slight pull down (aim at neck)
Bullet 6–10: Medium pull down
Bullet 11–20: Hard pull down + slight left-right adjust
Bullet 21–30: Full pull (crosshair is now below target)
```

**Training method:** Build a wall with colored blocks at 20m. Fire full mags at the blocks. Adjust your mouse pull until all bullets hit a 1-block area.

## 🏃 Movement & Positioning

### Movement Rules

| Rule | Why |
|:-----|:-----|
| **Always strafe** (A/D) while firing | Standing still = easy headshot |
| **ADAD spam** | Crouch-strafing throws off opponents |
| **Use cover** (rocks, trees, walls) | Never fight in the open |
| **Peek advantage** | Swing wide → fire → retreat behind cover |
| **Don't repeak the same angle** | Enemy will pre-aim it |
| **Jump peeking** | Quick jump over cover to spot enemies |
| **Crouch spam** | Disrupts headshot aim |

### Peeking Strategies

| Peek Type | When to Use | Risk |
|:----------|:------------|:-----|
| **Jiggle peek** | Check if someone is holding angle | Low — fast in/out |
| **Wide peek** | Commit to a fight, full damage trade | High — enemy can hit you |
| **Crouch peek** | Surprise the enemy (they aimed head height) | Medium |
| **Jump peek** | Spot enemies behind wall | Low (but predictable) |
| **Pre-fire peek** | Hold fire, round corner, fire immediately | Medium |

### Positioning Priority

1. **High ground** — you see more, harder to hit
2. **Cover** — tree, rock, base wall, monument structure
3. **Escape route** — never trap yourself in a dead end
4. **Crosshair at head height** — always. Walk with ADS at head level.
5. **Third-person advantage** — corners with open sides allow you to see without being seen

## 🧠 Mind Games & Strategy

| Tactic | How | When |
|:-------|:-----|:------|
| **Door camping** | Wait at a base door | When you know someone's inside |
| **Bait loot** | Drop a stack of scrap, ambush grabber | Anywhere |
| **Third-party** | Let two groups fight, clean up survivors | **Always** when possible |
| **Fake raiding** | Explosive noise near enemy base | Makes them panic-despawn |
| **Drop and swap** | Drop a med kit, bait enemy to pick it up | Close quarters |
| **Silent footstep** | Walk (don't run) to approach | Sneaking up on campers |
