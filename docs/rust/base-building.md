## Base Principles

Every base needs: TC coverage, honeycomb, airlock, stone or better walls.

## Popular Designs

2x1 Starter: 2 foundations, 1 door, TC in triangle. Raid cost: ~4 satchels.

2x2: 4 foundations + honeycomb. Garage door airlock. Raid cost: ~12 satchels.

Bunker Base: Drop-down bunker entrance. Raid cost: 16+ rockets.

Compound: Multiple bases inside high walls. External TCs. 30+ rockets.

## Honeycomb

Place triangle foundations around base. Upgrade to stone. Each layer = 6 walls raiders must breach.

## Doors

Garage doors on outer layer (cheaper). Armored doors on inner core. 5-7 doors between outside and loot.
