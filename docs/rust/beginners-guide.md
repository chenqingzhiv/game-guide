## First Hour

1. Gather 2 stones, craft Rock
2. Gather 500 wood + 200 stone for tools
3. Build a 2x1 base
4. Place sleeping bag
5. Craft wooden spear for defense

## First Day

- 0-1h: 2x1 base + bag
- 1-2h: Furnace
- 2-3h: Code lock
- 3-5h: Stone walls
- 5-8h: Bow + arrows
- 8-12h: Research BPs
- 12-24h: Expand base

## Scrap & BPs

- Workbench L1: 75 scrap
- Workbench L2: 500 scrap
- Workbench L3: 1250 scrap
- Priority: Code lock, garage door, metal tools, bow, revolver

## How to NOT Get Raided

1. Build away from monuments
2. Don't look rich
3. Stone walls (not wood)
4. Honeycomb
5. External TCs
6. Garage doors on outer layer
