# ✨ Minecraft Enchanting Guide

> Data version: Minecraft 1.21 | Last updated: 2026-07

Enchanting turns a good weapon into a legendary one. This guide covers everything from setting up a level 30 enchanting table to building XP farms, trading with librarians, and min-maxing your enchantments.

---

## 🔮 Enchanting Table Setup

### Minimum Level 30 Setup

To reach enchantment **level 30** (the maximum tier), you need **15 bookshelves** placed in a specific pattern around the enchanting table.

```
Bookshelf Layout (Top-Down View):
     
     B B B
   B X X X B
   B X E X B
   B X X X B
     B B B

E = Enchanting Table
B = Bookshelf (on any block)
X = Air gap (1 block between shelf and table)
```

### Bookshelf Placement Rules

| Rule | Detail |
|:-----|:--------|
| Distance | Exactly 1 block away (air gap) |
| Height | Bookshelves can be at same level or 1 block above table |
| Blocking | Torches, slabs, carpets between table and bookshelf? **Blocks connection** |
| Table level | Table must be on level. Bookshelf can be 1 above or same level. |

### Cost to Build

| Material | Quantity | How to Get |
|:---------|:---------|:-----------|
| Bookshelves | 15 | 3 books + 6 planks each = 45 books + 90 planks |
| Books | 45 | 3 paper + 1 leather each = 135 paper + 45 leather |
| Sugar cane | ~68 stalks | Grows on sand/water edge — farm 3×20 patch |
| Leather | 45 | Kill cows (3 per = 15 cows) or fish |

> 💡 **Faster alternative:** Break bookshelves in villages and take them! Most libraries have 5–8 shelves.

### 🏆 Best Early-Game XP Farm: Zombie Spawner

1. Find a zombie spawner in a cave (dark mossy cobblestone room)
2. Dig a 2×1 tunnel from spawner area
3. Drop zombies 20 blocks (10 hearts damage — they survive at 1 HP)
4. Hit them once with a Looting III sword for XP + optional rotten flesh
5. Light up surrounding caves to prevent mob interference

### 🏆 Best Endgame XP Farm: Enderman Farm

1. Go to The End main island
2. Build a platform at Y=250 (above endermen spawn area)
3. Create a chunk-aligned 30×30 platform at Y=0
4. Water streams funnel endermen into a 40+ block drop
5. Hit at feet level with sweeping edge sword
6. Collect pearls + XP

## 📖 Librarian Trading: Enchanted Books

Trading with villagers is often **faster** than enchanting tables for specific enchantments.

### Tier 1 Librarian Trades

| Trade | Cost | Items |
|:------|:-----|:------|
| Name tag | 20 emeralds | 1 book |
| Enchanted book | 5–64 emeralds | 1 book |

## 🔨 Anvil Mechanics

### Anvil Use Costs

| Operation | Base Cost | Minimum XP |
|:----------|:----------|:-----------|
| Rename item | 1 level | 1 |
| Apply enchanted book | 1–9 levels | 1 |
| Combine items | 2–20 levels | 2 |
| Repair (material) | 1–4 levels | 1 |

### Anvil Rules

| Rule | Detail |
|:-----|:--------|
| **Max anvil uses** | 6 (item becomes "Too Expensive!" after) |
| **Conflict table** | Can't combine Protection + Fire Protection on same item |
| **Priority** | Combine lower-level books first, then apply to tool |
| **Repair cost** | Prior work penalty doubles each time you use the anvil |
| **Better:** | Use Mending + XP instead of anvil repairs for tools you use often |

### Combining Enchantments Efficiently

```
BAD: Book (Sharpness I) → Apply to sword → Book (Sharpness II) → Apply
     Cost: 1 + 3 + 1 + 5 = 10 levels, 4 anvil uses

GOOD: Book (Sharpness I) + Book (Sharpness I) = Book (Sharpness II)
      Book (Sharpness II) + Book (Sharpness II) = Book (Sharpness IV)
      Apply Book (Sharpness IV) to sword
      Cost: 2 + 4 + 6 = 12 levels, 3 anvil uses
```

## ⚗️ Grindstone: Removing Enchantments

Use a **Grindstone** to remove all enchantments from an item. You get some XP back. Useful when:

- You accidentally put a bad enchantment on a good tool
- Enchanted books you found have conflicting enchantments
- You want to salvage an item before throwing it away
