## Design Principles

Depth: Add pillars, windowsills, recesses.
Texture: Mix similar blocks (stone + andesite).
Shape: Avoid boxes. Add roof overhangs, wings.
Color: 3-color palette max per build.
Scale: 11x11 rooms feel better than 7x7.

## Block Palettes

Medieval: Stone bricks + oak + dark oak roof.
Modern: White concrete + glass + birch.
Rustic: Oak logs + cobblestone + spruce.
Japanese: Spruce + bamboo + cherry.

Use stairs and slabs for roofs, not full blocks.

## Lighting

Torch (14): Rustic.
Lantern (15): Best all-round.
Sea Lantern (15): Modern, bright.
End Rod (14): Directional.

Place every 10 blocks in open areas, every 6 in hallways.
