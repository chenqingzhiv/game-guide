## What is Minecraft?

Best-selling game of all time (over 300 million copies). Open-world sandbox where you survive, build, craft, and explore procedurally generated worlds. From punching your first tree to defeating the Ender Dragon, every playthrough is unique.

### Why Minecraft Is Essential Reading

- **Creative freedom**: Build anything from a dirt hut to a fully automated castle
- **Massive modding ecosystem**: Thousands of mods extend gameplay infinitely
- **Cross-platform**: Play on Java or Bedrock with friends
- **Constant updates**: 1.22 snapshots bring new biomes, mobs, and mechanics

## Quick Start

1. **Day 1**: Punch wood → crafting table → wooden tools → shelter before night
2. **Day 2-5**: Expand shelter, make stone tools, find coal, start a farm
3. **Week 1**: Explore caves for iron, build bed, enchant table
4. **Progression**: Nether → Blaze rods → End portal → Ender Dragon → Elytra
