## First Hour

1. Collect iron, copper, limestone near drop pod
2. Build Portable Miner on iron node
3. Craft smelter > iron plates
4. Build biomass burner for power
5. Start producing iron rods + screws

Pro tip: Build near multiple resource nodes.

## Power Management

Biomass burners: Chop leaves/wood, convert to biomass.
Coal generators: Unlock at tier 3. 8 generators per 120 coal/min node.

Tip: Build a biomass farm (solid biofuel) before coal.

## Factory Layout

Input > Smelter > Constructor > Assembler > Storage.

Leave 2-3 foundation gaps between production lines for belting. Use manifolds for even splitting.
