## What is Core Keeper?

Mining sandbox survival game where you explore a vast underground world. Gather resources, build bases, fight giant bosses, and unlock automation systems. Think **Terraria meets Stardew Valley** with a focus on mining and exploration.

### Core Systems

| System | Description | Unlock |
|:-------|:------------|:-------|
| ⛏️ **Mining** | Dig through dirt, clay, stone, and slime | Start |
| 🏠 **Building** | Craft workstations, walls, furniture | Start |
| 👹 **Combat** | Melee, ranged, magic weapons | Start |
| ⚡ **Automation** | Conveyor belts, drills, smelters | Mid-game |
| 🌿 **Farming** | Grow crops, raise glowing animals | Mid-game |
| 🧪 **Crafting** | Anvils, workbenches, jewelry stations | Progression |

## Quick Progression

1. **Spawn → find your base/glow tulips**  
2. Mine copper/tin → bronze tools and weapons  
3. Find first boss (Glurch or Ghorm)  
4. Explore Clay Caves → Larva biome  
5. Find the Great Wall → Desert of Beginnings  
6. Unlock drills and automation for infinite resources
