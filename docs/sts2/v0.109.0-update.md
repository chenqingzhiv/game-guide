# Slay the Spire 2 v0.109.0 Update Guide — Biggest Patch Yet

> 更新日期: July 17, 2026 | 来源: Official Beta Patch Notes
> 前版: [v0.108.0 多人卡牌更新](v0.108.0-update.md)

## 概述

v0.109.0 是 Slay the Spire 2 迄今为止最大的补丁，包括 2 个全新 Neow 遗物、多张关键卡牌重做、以及大量数值平衡调整。

---

## 一、全新 Neow 遗物

### 探矿杖 (Dowsing Rod)

每局开始时获得一张"探矿"任务卡。访问 5 个 "?" 房间后，转化为 **丰饶 (Abundance)**：

> **丰饶** — 1 费技能。从 3 张升级过的能力牌中选择 1 张加入手牌，本回合 0 费打出。

**评价**: 相当于一个可控的遗物级能力牌获取器，适合需要特定能力的构筑路线。

### 涅奥的祭品 (Neow's Sacrifice)

开局获得一瓶 **龙涎香 (Ambergris)** 药水，但牌组加入一张 **内疚 (Guilty)** 诅咒。

> **龙涎香**: 回复 50% 最大生命值，并额外获得一个回合。

**评价**: 高风险高回报——诅咒会在关键战斗中卡手，但多一回合等于翻倍输出。

## 五、策略影响

1. **Silent 大幅增强** — 周密计划不再弃牌意味着 Silent 可以跨回合囤积关键卡，配合专精的手牌管理，Silent 的稳定性达到新高度。

2. **Necrobinder 新玩法** — 幻灵一次性打出所有虚灵牌，可以围绕虚灵体系构筑"爆发收割"路线。

3. **多人模式节奏变化** — 午夜和宴会球的削弱降低了多人对局的 burst 伤害上限，游戏更注重持续协作而非秒杀。

4. **Regent 需要调整思路** — 创世柱的削弱意味着不能再无脑叠甲，需要更精确地规划创造时机。
