# 📈 Shapez 2 Milestones & Upgrade Unlock Table

Each milestone you complete at the hub unlocks new buildings, technologies, and capabilities. This reference covers every milestone tier and what it gives you, so you can plan your factory progression efficiently.

> **Data source:** Shapez 2 in-game manual v0.9.0 — Last verified: July 2026

---

## How Milestones Work

Milestones are the primary progression system in Shapez 2. Each tier requires delivering specific shapes to the hub. Completing a milestone:

- ✅ Unlocks new **buildings** and **upgrades**
- ✅ Increases your **building cap**
- ✅ Extends your **map borders**
- ✅ Grants access to new **crystal types** on the map

Higher tier milestones require more complex shapes (more quadrants, colors, layers).

## Building Upgrade Progression

As you unlock higher building levels, your buildings process shapes faster:

### Building Speed by Level

| Level | Speed Multiplier | Unlock Tier |
|:------|:-----------------|:------------|
| 1 | 1× | Default |
| 2 | 1.5× | Tier 3 |
| 3 | 2× | Tier 5 |
| 4 | 3× | Tier 7 |
| 5 | 4× | Tier 9 |
| 6 | 6× | Tier 11 |
| 7 | 8× | Tier 13 |
| 8 | 10× | Tier 15 |
| 9 | 12× | Tier 17 |
| 10 | 15× | Tier 18 |
| 11 | 18× | Tier 21 |
| 12 | 22× | Tier 22 |

### MK Building Upgrades

| Upgrade | Effect | Unlock | Cost |
|:--------|:-------|:-------|:-----|
| **MK2** | ×2 throughput per building | Tier 19 | Shapes + Crystals |
| **MK3** | ×3 throughput (base ×6 total) | Tier 20 | More complex shapes |
| **MK4** | ×4 throughput (base ×8 total) | Late-game optional | Various |

MK buildings use the same footprint but process shapes significantly faster. They require a **separate research cost** at the hub — building them costs specific shape deliveries.

## Recommended Priority Path

### For Speedrunners

```
Tier 1-4 → Rush to unlock Mixer (Tier 6)
→ Build automated dye production
→ Rush to Stacker (Tier 3)
→ Focus on 2-layer colored shapes (Tier 13)
→ Logic gates (Tier 14+)
```

### For New Players

```
1. Complete Tier 1-4 without over-optimizing
2. Build a compact color factory producing Yellow, Cyan, White
3. Reach Tier 9 for 2-layer shapes
4. Start designing stamps for reusable production lines
5. Push to Tier 17 for 3-layer shape access
```

### Resource Management Tips

| **Mixed Deposits** | Tier 9+ | Efficiency: single miner → multiple colors |

## Hub Input Optimization

The hub accepts shapes from **any direction** (top, bottom, left, right). Tips:

- Use multiple input belts from different directions for high-throughput deliveries
- Buffer completed shapes in storage — belt directly from the storage to the hub
- One belt per shape type is usually sufficient until Tier 15+
