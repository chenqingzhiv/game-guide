## 🎯 What is Shapez 2?

Shapez 2 is a **pure factory automation puzzle** — no combat, no survival, no resource limits. Your goal is to extract geometric shapes, cut them apart, combine them, and deliver complex products to the hub.

**What makes it unique:**
- **Infinite, procedurally generated map** — never run out of space
- **No enemies, no pollution, no time pressure** — pure building
- **Deep logic puzzles** — combining shapes gets surprisingly complex
- **No resource draining** — miners produce infinitely

## 🔷 Shape Basics

| Shape Component | Description | Example |
|:----------------|:------------|:--------|
| **Quadrant** | 1/4 of a shape (top-left, top-right, bottom-left, bottom-right) | Quarter-circle, square |
| **Layer** | Each shape can have up to 4 layers | Colored on top, then white, etc. |
| **Color** | White, Red, Green, Blue, Yellow, Purple, Cyan | Combine primary colors to make secondary |
| **Pins/Wires** | Logic wires for binary signals | Compare, combine, filter |

## ⚙️ Production Progression

```
Tier 1: Extract Shapes     → Deliver basic shapes (circle, square)
Tier 2: Cut & Combine      → Split shapes, merge quadrants
Tier 3: Color Mixing       → Extract dyes, color shapes
Tier 4: Stacking           → Multi-layer shapes
Tier 5: Logic              → Pin/wire logic, conditional processing
Tier 6-10: Complex Targets → Full 4-layer colored shapes at high speed
```

## 🧊 Key Production Tips

| Tip | Why It Matters |
|:----|:---------------|
| **Build modular** | Copy-paste blueprint groups. Don't rebuild from scratch. |
| **Use stackers early** | Two simple shapes → one complex shape is the core mechanic. |
| **Color separate from shape** | Mix dye into clear shapes rather than recoloring complex ones. |
| **Leave bus room** | 8-12 lane main bus is enough for mid-game. |
| **Upgrade belts often** | Speed matters more than efficiency in Shapez 2. |
