## 隐藏任务技巧

### 1️⃣ 时间条件任务
```
部分任务只在特定时间段出现：
- 夜晚(18:00-6:00) → 暗夜英雄/夜叉之愿
- 白天(6:00-18:00) → 海上盛珠/海祇岛任务
建议：随时调时间试试
```

### 2️⃣ 多阶段任务
```
靖世九柱/古云有螭 等任务有3-5个阶段
中间可能需要等现实天数刷新
**不要卡在一阶段就放弃**
```

### 3️⃣ 成就关联
```
很多隐藏任务完成后会解锁成就：
- 天地万象系列
- 心跳的记忆
- 岩下之旅
做任务前先看看成就是否已完成
```
