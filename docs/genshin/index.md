# 原神 — 游戏概览

## 基础信息

- **开发商**: 米哈游
- **类型**: 开放世界RPG
- **平台**: PC / PS5 / PS4 / iOS / Android
- **月活**: 6000万+

## 核心系统

### 元素反应

| 元素组合 | 反应 | 效果 |
|---------|------|------|
| 火+水 | 蒸发 | 1.5x/2x伤害 |
| 火+雷 | 超载 | 爆炸AOE |
| 水+雷 | 感电 | 持续雷伤 |
| 火+冰 | 融化 | 1.5x/2x伤害 |
| 水+冰 | 冻结 | 控场 |
| 雷+冰 | 超导 | 减物抗40% |
| 风+其他 | 扩散 | 传播元素 |
| 岩+其他 | 结晶 | 护盾 |

### 树脂规划

**每天建议：**
- 地脉之花（摩拉/经验书）×4
- 周本BOSS ×3（减半体力）
- 天赋/武器突破材料（按需）
- 圣遗物本（后期再刷）

## 新手路线

1. **1-20级** — 推主线+开传送点
2. **20-35级** — 收集神瞳+世界任务
3. **35-45级** — 开始刷圣遗物
4. **45级+** — 大量刷5星圣遗物
