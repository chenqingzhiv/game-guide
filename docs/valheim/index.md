# Valheim ⚔️

**Survive, build, and conquer in a Viking purgatory.** From a simple lean-to to a thriving longhouse — master the meadows, survive the swamps, and challenge the Forsaken bosses.

## What is Valheim?

Valheim is a brutal exploration and survival game set in a procedurally-generated world inspired by Norse mythology. You play as a fallen Viking sent to the Tenth World, tasked with killing the Forsaken — ancient beasts that hold the secrets of the gods.

---

## Game Features

| Feature | Description |
|:--------|:------------|
| 🌍 **Open World** | Procedurally generated biomes — Meadows, Black Forest, Swamp, Mountains, Plains, Mistlands, Ashlands |
| ⚔️ **Combat** | Melee, ranged, and magic — timed blocks, parries, and dodges |
| 🏗️ **Building** | Full structural physics — walls, roofs, supports, and terraforming |
| 🛠️ **Crafting** | Tiered workbenches, forges, and crafting stations |

## Getting Started Guide

### 1. First Steps

1. Spawn in the **Meadows**
2. Gather **wood** and **stone** by hand
3. Build a basic **shelter** (workbench + bed + fire)
4. Craft **basic tools** (axe, hammer, pickaxe)
5. Hunt **boar** and **deer** for leather and meat

### 2. Progression Path

```
Meadows (Eikthyr) 
  → Black Forest (The Elder) 
    → Swamp (Bonemass) 
      → Mountains (Moder) 
        → Plains (Yagluth) 
          → Mistlands (Queen) 
            → Ashlands (Fader)
```

Each biome requires better gear, food, and preparations before venturing in.

### 3. Essential Crafting Stations

| Station | Upgrade Needed For |
|:--------|:------------------|
| Workbench | Tools, weapons, buildings — upgrade with chopping block, tanning rack |
| Forge | Metal weapons, armor, tools — upgrade with anvils, coolers |
| Cauldron | Advanced food recipes — upgrade with spice rack, butcher table |
| Blast Furnace | Black metal, black marble processing |
| Windmill / Spinning Wheel | Flax → linen thread |

## Food & Combat Tips

- **Never eat just one food** — eat 3 different foods for max HP + stamina
- **HP food**: Cooked meat, fish wraps, lox meat pie
- **Stamina food**: berries, mushrooms, honey, bread
- **Balanced**: turnip stew, wolf skewer, serpent stew
- **Parry is king** — a well-timed block + parry opens enemies for bonus damage
- **Poison resist mead** is mandatory before entering the Swamp

---

## Building Tips

- **Dig a moat** around your base — most enemies can't cross deep water
- **Raise ground** with a hoe to create natural walls
- **Core wood poles** allow taller buildings
- **Stonecutter** unlocks stone buildings (more durable, fire-proof)
- **Smoke** needs a chimney — build an open peak or vent
