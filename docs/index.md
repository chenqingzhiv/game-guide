## 新手指南

??? tip "我也是新手，从哪开始？"

    本站所有内容均为 **数据库式整理**，方便你随时查阅：
    
    - **装备数据库** — 按类型/等级/属性筛选，找到最适合你的装备
    - **角色攻略** — 技能搭配 / 流派推荐 / 加点方案
    - **地图标记** — 隐藏宝箱 / 稀有怪 / 任务NPC位置
    - **任务流程** — 分支选择 / 奖励对比 / 隐藏结局
