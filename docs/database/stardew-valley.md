# 星露谷物语 Stardew Valley

由 Eric Barone（ConcernedApe）开发的经典农场模拟游戏。

---

## ⏱️ 快速入门

- **季节**：春季 → 夏季 → 秋季 → 冬季（28天/季）
- **赚钱核心**：初期钓鱼 → 中期酿酒 → 后期上古水果
- **好感度**：对话（+20）→ 送礼（+45~+80）→ 结婚（需要10心）
