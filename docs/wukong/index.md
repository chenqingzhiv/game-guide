# 黑神话：悟空 — 游戏概览

## 基本信息

- **开发商**: 游戏科学（Game Science）
- **发售日**: 2024年8月20日
- **平台**: PC / PS5 / Xbox Series X|S
- **类型**: 动作角色扮演（ARPG）
- **全球销量**: 超3000万份

## 系统介绍

### 战斗系统

黑悟空的战斗核心是 **棍法+法术+变身** 的三位一体系统：

| 系统 | 说明 | 获取方式 |
|------|------|---------|
| 棍法 | 劈棍/立棍/戳棍三种架势 | 初始解锁 |
| 法术 | 定身术/毫毛术/铜头铁臂等 | 主线推进 |
| 变身 | 变身狼/龙/猿等 | 击败特定BOSS |
| 法宝 | 绣花针/芭蕉扇/金箍 | 支线任务 |

### 装备系统

- **武器**: 棍棒，可升级强化
- **防具**: 头冠/衣甲/臂甲/腿甲
- **珍玩**: 提供特殊被动效果

## 新手建议

1. 优先升级**气力**和**生命**
2. 学会**识破**（完美闪避/招架）可大幅降低难度
3. 多探索地图，隐藏区域往往有好东西
4. 每个BOSS都有弱点，多试几次找规律
