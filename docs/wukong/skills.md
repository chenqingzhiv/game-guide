## 推荐Build方案

### 🟢 新手通用Build（推荐）

```
加点顺序：
1. 闪身Lv3（生存第一）
2. 劈棍Lv3（主力输出）
3. 棍花Lv2（攒气）
4. 体力上限Lv2（续航）
5. 气力恢复Lv2
```

**适合：** 前期开荒，boss战

### 🔴 暴力输出Build

```
加点顺序：
1. 劈棍Lv5（伤害最大化）
2. 暴击率Lv3（配合劈棍）
3. 攻击力Lv3（面板叠高）
4. 棍花Lv2（快速攒气）
5. 闪身Lv1（保底生存）
```

**适合：** 熟悉操作后的玩家

### 🟣 生存控制Build

```
加点顺序：
1. 闪身Lv5（完美闪避）
2. 立棍Lv3（范围控场）
3. 体力Lv3（翻滚不喘）
4. 防御Lv3（减伤）
5. 戳棍Lv2（远程偷刀）
```

**适合：** boss战卡关时

## 天赋搭配

| 天赋 | 效果 | 适配流派 |
|:----|:-----|:--------|
| 🔥 火属性附魔 | 攻击附加火伤 | 暴力输出 |
| ❄️ 冰属性附魔 | 减速敌人 | 生存控制 |
| ⚡ 雷属性附魔 | 额外暴击率 | 暴击流 |
| 🌪️ 风属性附魔 | 破防效果 | 全能 |

> **铁律：** 无论如何至少留1点给闪身。没有闪身，后面几个boss根本没法打。
