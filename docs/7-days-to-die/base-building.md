## Base Types

Starter: 5x5 wood with 2-block walls. Upgrade to cobblestone by Day 7.

Horde Base: Separate from main base. Kill corridor with blade traps + electric fence.

Mega Base: Concrete walls + drawbridge + moat. 10+ blocks thick.

## Defense Layers

Outer: Barbed wire + spikes (slows zombies).
Mid: Electric fence (stuns).
Inner: Blade traps + auto-turrets.
Core: Reinforced hatch + fallback bunker.

Always have an escape tunnel!

## Upgrades

Wood > Cobblestone > Concrete > Steel

Reinforced doors: Iron > Vault > Garage

Repair between hordes. Upgrade one layer at a time.
