## Satisfactory 1.1 Patch — Power Rebalance and Blueprint 2.0

Coffee Stain Studios has shipped **Satisfactory 1.1**, the first major content patch since the game's 1.0 launch. The update focuses on three pillars: **power system rebalancing**, **quality-of-life improvements** across the factory-building experience, and a significant expansion to the **Blueprint Designer** system. The patch is available now on Steam and Epic Games Store.

### Power System Overhaul

The 1.1 patch introduces the most significant changes to Satisfactory's power grid since the game entered early access.

**Nuclear Power Rebalance:**

- **Uranium Fuel Rods** — Energy output reduced from 7500 MJ per rod to 6000 MJ. This brings nuclear more in line with alternative endgame power options.
- **Plutonium Fuel Rods** — Output increased from 12,000 MJ to 14,000 MJ. Plutonium recycling is now more worthwhile as an endgame sink.
- **Ficsonium Fuel Rods** — Added as a new late-game fuel type. Requires Ficsite as a component. Produces 20,000 MJ per rod with zero waste. Crafting these is expensive, demanding the factory's full production chain.

**Geothermal Buff:**

Geothermal generators now produce **25% more power** (from 200 MW to 250 MW per generator on a pure node). Geyser locations have been updated on the world map — 5 new geyser nodes have been added:

| Location | Node Quality | Output |
|----------|-------------|--------|
| Northern Forest (W cliff) | Impure | 50 MW |
| Rocky Desert (canyon) | Normal | 125 MW |
| Swamp (deep) | Pure | 250 MW |
| Dune Desert (W edge) | Pure | 250 MW |
| Spire Coast (underwater cave) | Pure | 250 MW |

**Power Storage:**

Battery storage capacity increased from 100 MWh to **150 MWh** per unit. A new **Industrial Battery** building (unlocked at Tier 6) stores **500 MWh** and takes up a 2x2 footprint.

### Blueprint 2.0 — Copy-Paste for the Masses

The Blueprint Designer has been reworked from the ground up:

**Larger blueprints:**

- Blueprint Designer Mk.1 — 2x2x2 (unchanged, now tier 3 tech)
- Blueprint Designer Mk.2 — **4x4x4** (new, tier 5 tech)
- Blueprint Designer Mk.3 — **6x6x6** (new, tier 8 tech)

**Blueprint library:**
Blueprints are now stored in a **nested folder system**. You can organize blueprints by factory module, type, or biome. The library supports:
- Search by name or component tags
- Preview thumbnails (rendered in-engine)
- Version history (last 5 saves per blueprint)
- Import/export as `.sbp` files for sharing on the Satisfactory blueprint portal

**New blueprint tools:**
- **Rotate on import** — Rotate entire blueprints before placing, including internal belt orientations.
- **Mirror mode** — Mirror a blueprint along X, Y, or Z axis.
- **Component count** — Hover over a blueprint to see total resource cost and power consumption before building.

### Quality of Life Improvements

**Logistics:**

- **Programmable Splitter** — Now has a "Default" output option. Items not matching any rule go to the default output instead of backing up.
- **Smart Splitter** — Added an "Overflow Only" mode that sends items to the overflow belt when the primary output is blocked.
- **Train timetables** — Stations can now be saved as named presets and applied to multiple trains.
- **Fluid buffers** — Now output at 3x the previous rate, fixing the "slug flow" issue with large pipe networks.

**Building and Deconstruction:**

- **Mass deconstruction filter** — Hold Ctrl when using the deconstruction tool to filter by building type (e.g., only remove conveyor belts, leave constructors intact).
- **Hologram snapping** — Buildings now snap to world grid when holding Shift, even when not aligned to an existing foundation.
- **Buildable Pipeline Support Mk.2** — A new variant of the pipeline support that allows two pipelines side by side. Unlocked at Tier 4.

**UI/UX:**

- **Production graph** — A new tab in the Statistics panel shows power consumption over time (last 1 hour, 6 hours, or 24 hours).
- **Map markers** — Added color-coding for marker types (resource nodes, factories, exploration goals).
- **Codex search** — The in-game codex now supports fuzzy search with partial matching.
